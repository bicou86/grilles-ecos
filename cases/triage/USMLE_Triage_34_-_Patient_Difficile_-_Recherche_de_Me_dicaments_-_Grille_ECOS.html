Théorie et Pratique - Drug-seeking behavior

Reconnaissance du comportement drug-seeking

Patients développent techniques sophistiquées : réticence feinte, faire nommer médicament, exclusivité traitement, incohérences cliniques. Évaluation systématique nécessaire.

Tests cliniques de simulation

Tests de Waddell, incohérences Lasègue selon position, amplitude mouvement normale malgré douleur sévère. Documentation précise essentielle.

Gestion thérapeutique

Éviter prescription opioïdes, alternatives non-addictives, référence appropriée, documentation complète. Focus sur sécurité et continuité soins.

Examens complémentaires

  • Radiographies lombaires (évaluation structurelle)
  • Vérification base données prescriptions
  • Contact médecin traitant habituel
  • Tests de simulation (Waddell)
  • IRM seulement si justification clinique claire

Rappels thérapeutiques

  • Éviter prescription opioïdes si suspicion drug-seeking
  • Alternatives : AINS, relaxants musculaires, physiothérapie
  • Documentation détaillée comportement et incohérences
  • Référence médecin traitant pour continuité
  • Information patient sur alternatives thérapeutiques
Red flags du comportement drug-seeking

Demandes spécifiques

  • Demande médicament précis par nom
  • Fait nommer médicament par médecin
  • Prétend que seul un médicament fonctionne
  • Connaît dosages et noms commerciaux

Comportement paradoxal

  • Prétend réticence aux médicaments
  • Dit avoir haute tolérance douleur
  • Évite habituellement médecins
  • Urgence pour problème chronique

Incohérences cliniques

  • Symptômes disproportionnés à l'examen
  • Tests neurologiques inconsistants
  • Amplitude mouvement normale malgré douleur
  • Comportement non cohérent avec douleur rapportée

Histoire suspecte

  • Liste exhaustive échecs thérapeutiques
  • Allergies convenables aux alternatives
  • Médecin habituel indisponible
  • Imagerie antérieure non disponible
