Rappels théoriques - Diabète de type 2

Critères diagnostiques du diabète

Glycémie à jeun ≥ 126 mg/dL (7 mmol/L) à 2 reprises OU Glycémie aléatoire ≥ 200 mg/dL (11.1 mmol/L) + symptômes OU HbA1c ≥ 6.5% OU HGPO 2h ≥ 200 mg/dL. Prédiabète : Glycémie à jeun 100-125 mg/dL ou HbA1c 5.7-6.4%.

Symptômes cardinaux du diabète

Polyurie (diurèse osmotique par glycosurie), Polydipsie (compensation déshydratation), Polyphagie (déficit utilisation glucose cellulaire), Perte de poids (lipolyse, protéolyse, déshydratation). Fatigue par déficit énergétique cellulaire.

Complications aiguës

Hyperglycémie simple : symptômes classiques. Acidocétose diabétique : plus fréquente type 1 mais possible type 2. État hyperosmolaire : déshydratation sévère, altération conscience, mortalité élevée. Hypoglycémie iatrogène sous traitement.

Rappels thérapeutiques

  • Diabète nouvellement diagnostiqué : Metformine première ligne sauf contre-indication
  • Objectifs : HbA1c < 7% général, individualisé selon patient
  • Mesures hygiéno-diététiques fondamentales : perte 5-10% poids
  • Dépistage complications dès diagnostic : rétinopathie, néphropathie, neuropathie
  • Contrôle facteurs risque CV : HTA < 130/80, LDL < 70 mg/dL si diabète
  • Vaccinations : grippe annuelle, pneumocoque
  • Éducation thérapeutique : auto-surveillance, hypoglycémie, soins pieds

Examens complémentaires

  • Glycémie capillaire : dépistage immédiat, valeur > 200 mg/dL suggestive
  • Glycémie veineuse à jeun : gold standard diagnostic, 8h jeûne minimum
  • HbA1c : reflet équilibre 3 mois, diagnostic et suivi
  • Bilan rénal : créatinine, DFG, microalbuminurie/créatininurie
  • Bilan lipidique complet : fréquent syndrome métabolique
  • TSH : éliminer hyperthyroïdie (DD perte poids + polyphagie)
  • ECG : dépistage cardiopathie silencieuse

      • • Polyurie, polydipsie, fatigue, perte de poids
      • • Infections à répétition (mycoses, infections urinaires)
      • • Troubles visuels, paresthésies
      • • Dépistage souvent fortuit : bilan systématique, hyperglycémie isolée
      • Évaluer les facteurs de risque :
      • • Surpoids/obésité, sédentarité
      • • ATCD familiaux de diabète
      • • Hypertension, dyslipidémie, antécédents cardiovasculaires
      • • Antécédents de diabète gestationnel ou de macrosomie
      • Rechercher les complications déjà présentes :
      • • Neuropathies, troubles de la vision, douleurs des membres inférieurs, claudication, infections cutanées

  • Titre : 👀 Examen clinique

    Subsections :

    • Titre : 🔍 Ce qu'il faut rechercher :

      Points :
      • Poids, IMC, tour de taille : identifier syndrome métabolique
      • TA : fréquente association HTA
      • Examen cardiovasculaire : souffle, pouls périphériques, abolition des pouls
      • Pieds :
      • • Inspection : plaies, mycoses, déformation
      • • Palpation artérielle : pouls tibial/postérieur
      • • Sensibilité : test au monofilament, diapason (neuropathie)
      • Fond d'œil (si possible ou adressé à l'ophtalmologue)

  • Titre : 🧪 Diagnostic

    Subsections :

    • Titre : 🧬 Critères biologiques :

      Points :
      • Glycémie à jeun ≥ 126 mg/dL (≥ 7,0 mmol/L) ×2
      • Glycémie aléatoire ≥ 200 mg/dL avec symptômes
      • HbA1c ≥ 6,5 % (si méthode certifiée NGSP)
      • HGPO 75 g : glycémie à 2h ≥ 200 mg/dL
      • ⚠️ Refaire le test si asymptomatique pour confirmer le diagnostic.

  • Titre : ⚙️ Prise en charge

    Subsections :

    • Titre : 🥗 Mesures non pharmacologiques (1ʳᵉ ligne)

      Points :
      • Perte de poids (objectif : ≥ 5–10 %)
      • Alimentation équilibrée (régime méditerranéen, hypoglucidique modéré)
      • Activité physique régulière (150 min/semaine, combinée)
      • Sevrage tabagique, réduction alcool

    • Titre : 💊 Traitement médicamenteux

      Tableau :
      Colonnes :
      • Stade
      • Traitement
      Lignes :

      • 0 : 1ʳᵉ ligne

        1 : Metformine si non contre-indiquée

      • 0 : Si HbA1c > 8,5 % ou comorbidités

        1 : Ajout : inhibiteur SGLT2 ou agoniste GLP-1

      • 0 : Si insuffisance cardiaque / maladie rénale

        1 : Priorité aux SGLT2 (empagliflozine, dapagliflozine)

      • 0 : Si risque CV élevé

        1 : GLP-1 (liraglutide, semaglutide) ou SGLT2

      • 0 : Si échec bi/thérapie

        1 : Ajouter insuline basale ± autres antidiabétiques

      • 0 : Attention

        1 : Surveillance hypoglycémies (notamment si sulfamides, insuline)

    • Titre : 🎯 Objectif glycémique :

      Points :
      • HbA1c < 7 % en général
      • Individualiser selon âge, comorbidités, espérance de vie

  • Titre : ✅ Points clés ECOS

    Points :
    • Penser au dépistage actif chez patients à risque (surpoids, ≥ 45 ans…).
    • Bien connaître les critères diagnostiques et les bilans initiaux.
    • Toujours inclure l'examen des pieds et la recherche de neuropathie.
    • Adapter la prise en charge selon le risque cardiovasculaire et rénal.
    • Savoir éduquer le patient (rôle central du patient dans la gestion).
